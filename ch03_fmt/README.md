# Chapter 03: fmt 패키지를 이용한 텍스트 입출력

Go 프로그래밍에서 가장 많이 사용하는 `fmt` 패키지의 입출력 기능을 배운다. `fmt` 패키지는 거의 모든 Go 프로그램에서 사용되며, 데이터를 화면에 출력하거나 키보드로 입력받는 기본적인 기능을 제공한다.

---


## 3.1 표준 입출력

`fmt` 패키지는 **format**의 약자로, 서식에 맞춰 데이터를 입출력하는 기능을 제공한다. C 언어의 `printf`, `scanf`에서 영감을 받았지만, Go만의 간결함과 타입 안전성을 추가하였다.

### 출력 함수 3가지

| 함수 | 설명 |
|------|------|
| `fmt.Print()` | 값을 출력한다 (줄바꿈 없음) |
| `fmt.Println()` | 값을 출력하고 줄바꿈한다 |
| `fmt.Printf()` | 서식(format)에 맞춰 출력한다 |

각 함수는 용도가 다르므로, 상황에 맞게 선택하여 사용해야 한다. `Println`은 디버깅이나 간단한 출력에, `Printf`는 정밀한 서식 제어가 필요할 때 사용한다.

### fmt.Println()

가장 자주 쓰는 출력 함수이다. 값을 출력한 후 **자동으로 줄바꿈**한다.

```go
fmt.Println("Hello, World!")        // 문자열 출력
fmt.Println(42)                     // 숫자 출력
fmt.Println("나이:", 25, "살")       // 여러 값 (공백으로 구분된다)
fmt.Println()                       // 빈 줄 출력
```

`Println`은 여러 인자를 전달하면 각 인자 사이에 자동으로 공백을 삽입한다. 이 점이 `Print`와의 중요한 차이점이다. 또한 `Println`은 출력한 바이트 수와 에러를 반환하지만, 보통은 이 반환값을 무시한다.

### fmt.Print()

줄바꿈 없이 출력한다.

```go
fmt.Print("Hello, ")
fmt.Print("World!")  // 같은 줄에 이어서 출력된다
fmt.Print("\n")      // 직접 줄바꿈해야 한다
```

`Print`는 인자가 모두 문자열이 아닌 경우에만 인자 사이에 공백을 삽입한다. 문자열 인자끼리는 공백 없이 바로 이어 붙인다.

### fmt.Printf()

**서식 문자열(format string)**을 사용하여 출력 형식을 지정한다. 가장 유연한 출력 함수이다.

```go
name := "홍길동"
age := 25
height := 175.5

fmt.Printf("이름: %s\n", name)
fmt.Printf("나이: %d세\n", age)
fmt.Printf("키: %.1fcm\n", height)
fmt.Printf("이름: %s, 나이: %d세, 키: %.1fcm\n", name, age, height)
```

`Printf`는 줄바꿈을 자동으로 추가하지 않으므로, 줄바꿈이 필요하면 `\n`을 직접 넣어야 한다. 서식 지정자의 개수와 인자의 개수가 일치하지 않으면 예상치 못한 출력이 나올 수 있으므로 주의해야 한다.

### 서식 지정자 (Format Verbs) 총정리

#### 일반

| 서식 | 설명 | 예시 |
|------|------|------|
| `%v` | 기본 형식으로 값을 출력한다 | `fmt.Printf("%v", 42)` → `42` |
| `%+v` | 구조체 필드 이름을 포함하여 출력한다 | `{Name:Go Age:15}` |
| `%#v` | Go 문법 형태로 출력한다 | `main.Person{Name:"Go", Age:15}` |
| `%T` | 타입을 출력한다 | `fmt.Printf("%T", 42)` → `int` |
| `%%` | % 기호 자체를 출력한다 | `fmt.Printf("100%%")` → `100%` |

`%v`는 "value"의 약자로, 어떤 타입이든 기본적인 형태로 출력해 주므로 디버깅 시 매우 유용하다. `%T`는 변수의 타입을 확인할 때 자주 사용한다.

#### 정수

| 서식 | 설명 | 예시 |
|------|------|------|
| `%d` | 10진수 | `42` |
| `%b` | 2진수 | `101010` |
| `%o` | 8진수 | `52` |
| `%x` | 16진수 (소문자) | `2a` |
| `%X` | 16진수 (대문자) | `2A` |
| `%c` | 유니코드 문자 | `*` (42) |

#### 실수

| 서식 | 설명 | 예시 |
|------|------|------|
| `%f` | 소수점 표기 | `3.141593` |
| `%e` | 지수 표기 (소문자) | `3.141593e+00` |
| `%E` | 지수 표기 (대문자) | `3.141593E+00` |
| `%g` | 짧은 표기 자동 선택 | `3.14159` |
| `%.2f` | 소수점 2자리 | `3.14` |

`%g`는 값의 크기에 따라 `%e`와 `%f` 중 더 짧은 형태를 자동으로 선택한다. 불필요한 후행 0도 제거해 주므로 깔끔한 출력이 필요할 때 유용하다.

#### 문자열

| 서식 | 설명 | 예시 |
|------|------|------|
| `%s` | 문자열 | `Hello` |
| `%q` | 큰따옴표로 감싼 문자열 | `"Hello"` |
| `%x` | 16진수 바이트 | `48656c6c6f` |

`%q`는 문자열 안에 특수 문자가 포함되어 있을 때 이스케이프 형태로 보여주므로, 디버깅 시 공백이나 제어 문자를 확인할 때 유용하다.

#### 불리언

| 서식 | 설명 | 예시 |
|------|------|------|
| `%t` | true 또는 false | `true` |

### 너비와 정렬

```go
// 최소 너비 지정
fmt.Printf("[%10d]\n", 42)      // [        42]  (오른쪽 정렬, 너비 10)
fmt.Printf("[%-10d]\n", 42)     // [42        ]  (왼쪽 정렬)
fmt.Printf("[%010d]\n", 42)     // [0000000042]  (0으로 채움)

// 문자열 너비
fmt.Printf("[%20s]\n", "Hello") // [               Hello]
fmt.Printf("[%-20s]\n", "Hello")// [Hello               ]

// 실수 너비와 정밀도
fmt.Printf("[%10.2f]\n", 3.14)  // [      3.14]
```

너비와 정렬은 표 형태의 데이터를 출력할 때 매우 유용하다. `-` 플래그는 왼쪽 정렬, `0` 플래그는 빈 공간을 0으로 채운다. `10.2f`에서 `10`은 전체 너비, `.2`는 소수점 이하 자릿수를 의미한다.

## 3.2 표준 입력

`fmt` 패키지는 키보드로부터 입력을 받는 함수도 제공한다.

### 입력 함수 3가지

| 함수 | 설명 |
|------|------|
| `fmt.Scan()` | 공백/줄바꿈으로 구분하여 입력받는다 |
| `fmt.Scanf()` | 서식에 맞춰 입력받는다 |
| `fmt.Scanln()` | 한 줄에서 공백으로 구분하여 입력받는다 |

### fmt.Scan()

공백이나 줄바꿈을 구분자로 사용하여 값을 입력받는다.

```go
var name string
var age int

fmt.Print("이름과 나이를 입력하세요: ")
fmt.Scan(&name, &age)  // & : 변수의 주소를 전달한다

fmt.Printf("이름: %s, 나이: %d\n", name, age)
```

> **`&` (앰퍼샌드)**: 변수의 메모리 주소를 전달한다. Scan 함수는 이 주소를 통해 변수에 직접 값을 저장한다. `&`를 빼먹으면 컴파일 에러가 발생하므로 반드시 붙여야 한다. 포인터에 대해서는 이후 장에서 자세히 배운다.

`Scan` 함수는 성공적으로 읽은 항목의 개수와 에러를 반환한다:

```go
n, err := fmt.Scan(&name, &age)
fmt.Printf("%d개 항목을 읽었다\n", n)
if err != nil {
    fmt.Println("입력 에러:", err)
}
```

### fmt.Scanf()

서식 문자열에 맞춰 입력을 받는다.

```go
var year, month, day int

fmt.Print("날짜를 입력하세요 (YYYY/MM/DD): ")
fmt.Scanf("%d/%d/%d", &year, &month, &day)

fmt.Printf("%d년 %d월 %d일\n", year, month, day)
```

> **주의**: `Scanf`는 운영체제에 따라 동작이 다를 수 있어, 실무에서는 `Scan`이나 `Scanln`을 더 많이 사용한다. 특히 Windows에서 줄바꿈 처리 방식이 다르기 때문에 예상치 못한 동작이 발생할 수 있다.

### fmt.Scanln()

한 줄 안에서 공백으로 구분하여 입력받는다. **줄바꿈이 나오면 입력을 중단**한다.

```go
var firstName, lastName string

fmt.Print("성과 이름을 입력하세요: ")
fmt.Scanln(&firstName, &lastName)

fmt.Printf("성: %s, 이름: %s\n", firstName, lastName)
```

### Scan vs Scanln 차이점

```go
// fmt.Scan: 줄바꿈을 공백처럼 취급한다
// 입력: "홍" [Enter] "길동" [Enter]  → 정상 동작

// fmt.Scanln: 같은 줄에서만 입력을 받는다
// 입력: "홍 길동" [Enter]  → 정상 동작
// 입력: "홍" [Enter]       → 첫 번째 값만 받고 종료
```

이 차이는 실제로 프로그램을 작성할 때 중요하다. 사용자가 한 줄에 모든 값을 입력하기를 기대한다면 `Scanln`을, 여러 줄에 걸쳐 입력해도 되도록 하려면 `Scan`을 사용한다.

## 3.3 키보드 입력과 Scan() 함수의 동작 원리

### 표준 입력 버퍼 (stdin buffer)

키보드 입력은 바로 프로그램으로 전달되지 않고, **표준 입력 버퍼(stdin buffer)**에 임시 저장된다.

```
키보드 → [표준 입력 버퍼] → Scan() 함수 → 변수에 저장
```

이 구조를 이해하는 것이 중요한 이유는, 버퍼에 남은 데이터가 다음 입력 함수 호출에 영향을 줄 수 있기 때문이다.

### 동작 과정

1. 사용자가 키보드로 `"25 홍길동"` 입력 후 Enter
2. 입력 버퍼에 `"25 홍길동\n"`이 저장된다
3. `Scan(&age)` 호출 → 버퍼에서 `"25"` 읽고 `int`로 변환하여 `age`에 저장한다
4. `Scan(&name)` 호출 → 버퍼에서 `"홍길동"` 읽고 `name`에 저장한다
5. `\n`은 구분자로 사용되고 버려진다

### 버퍼에 남는 데이터 문제

```go
var a int
var b int

fmt.Print("첫 번째 숫자: ")
fmt.Scanln(&a)    // 입력: "10 20" → a=10, "20\n"이 버퍼에 남는다

fmt.Print("두 번째 숫자: ")
fmt.Scanln(&b)    // 버퍼에 남은 "20"을 읽어서 b=20이 된다 (입력 없이!)
```

이 문제는 초보자가 가장 자주 겪는 함정 중 하나이다. 특히 `Scanln`을 연속으로 호출할 때 발생하기 쉽다.

### bufio.Scanner를 사용한 안전한 입력

실무에서는 `fmt.Scan` 대신 `bufio.Scanner`를 사용하는 것이 더 안전하다:

```go
import (
    "bufio"
    "fmt"
    "os"
)

scanner := bufio.NewScanner(os.Stdin)

fmt.Print("한 줄 입력: ")
scanner.Scan()
line := scanner.Text()
fmt.Println("입력값:", line)
```

`bufio.Scanner`는 한 줄 전체를 문자열로 읽어오므로, 버퍼 문제가 발생하지 않는다. 읽어온 문자열을 `strconv.Atoi()` 등으로 원하는 타입으로 변환하여 사용하면 된다.

```go
scanner := bufio.NewScanner(os.Stdin)

fmt.Print("나이를 입력하세요: ")
scanner.Scan()
age, err := strconv.Atoi(scanner.Text())
if err != nil {
    fmt.Println("숫자를 입력해 주세요")
    return
}
fmt.Printf("입력한 나이: %d\n", age)
```

### Sprintf와 Fprintf

`fmt` 패키지에는 출력 대상이 다른 함수들도 있다:

| 함수 | 출력 대상 |
|------|----------|
| `Printf` | 표준 출력 (화면) |
| `Sprintf` | 문자열로 반환한다 (화면에 출력하지 않는다) |
| `Fprintf` | 지정된 Writer (파일 등)에 출력한다 |

```go
// Sprintf: 서식을 적용한 문자열을 생성한다
msg := fmt.Sprintf("이름: %s, 나이: %d", "홍길동", 25)
fmt.Println(msg)  // "이름: 홍길동, 나이: 25"

// Fprintf: 파일이나 다른 출력 대상에 쓴다
fmt.Fprintf(os.Stderr, "에러: %s\n", "문제 발생!")
```

`Sprintf`는 문자열을 동적으로 조합할 때 매우 자주 사용된다. 로그 메시지를 만들거나, 에러 메시지를 생성하거나, 파일 경로를 조합할 때 등 다양한 상황에서 활용할 수 있다.

## 핵심 요약

| 항목 | 설명 |
|------|------|
| `Println` | 값 출력 + 줄바꿈이다 |
| `Printf` | 서식 지정 출력이다 (`%d`, `%s`, `%f` 등) |
| `Scan` | 공백/줄바꿈 구분자로 입력받는다 |
| `Scanln` | 같은 줄에서만 입력받는다 (줄바꿈 시 중단) |
| `Sprintf` | 서식 문자열을 생성한다 (출력하지 않는다) |
| `&변수` | 변수의 메모리 주소를 전달한다 |

## 연습문제

1. 이름, 나이, 키를 입력받아 "이름은 OOO이고, 나이는 OO세이며, 키는 OOO.Ocm이다."로 출력하는 프로그램을 작성하라.
2. 정수를 입력받아 10진수, 2진수, 8진수, 16진수로 모두 출력하는 프로그램을 작성하라.
3. `fmt.Sprintf`를 사용하여 "2024년 01월 01일" 형식의 날짜 문자열을 만드는 프로그램을 작성하라 (월과 일은 2자리로 0을 채워서 표시한다).
4. 두 개의 실수를 입력받아 합, 차, 곱, 몫을 각각 소수점 2자리까지 출력하는 프로그램을 작성하라.
5. `%v`, `%+v`, `%#v`의 차이를 직접 코드로 확인하라. 구조체를 하나 정의하고 세 가지 서식 지정자로 각각 출력하여 결과를 비교하라. (구조체는 `type Person struct { Name string; Age int }` 형태로 정의할 수 있다.)
6. `fmt.Printf`의 너비 지정 기능을 사용하여 아래와 같은 정렬된 표를 출력하는 프로그램을 작성하라:
   ```
   이름         나이    키
   홍길동        25    175.5
   김철수        30    180.2
   이영희        22    163.8
   ```
7. `fmt.Scanln`을 두 번 연속 호출하여 각각 다른 값을 입력받는 프로그램을 작성하라. 버퍼 문제가 발생하는 상황을 만들어 보고, `bufio.Scanner`로 해결하라.
8. `fmt.Sprintf`를 활용하여 "001", "002", ..., "010"과 같이 3자리 숫자 문자열을 생성하는 프로그램을 작성하라. (`%03d` 서식 지정자를 활용한다.)
9. `fmt.Fprintf`를 사용하여 표준 출력(`os.Stdout`)과 표준 에러(`os.Stderr`)에 각각 다른 메시지를 출력하는 프로그램을 작성하라. 터미널에서 `2>/dev/null`로 실행하면 어떤 메시지만 보이는지 확인하라.
10. `%q` 서식 지정자를 사용하여 빈 문자열, 공백이 포함된 문자열, 탭이 포함된 문자열, 줄바꿈이 포함된 문자열을 각각 출력하라. `%s`와 비교하여 어떤 차이가 있는지 설명하라.

## 구현 과제

1. **간단한 계산기**: 두 수와 연산자(+, -, *, /)를 입력받아 계산 결과를 출력하는 프로그램을 작성하라. `fmt.Scan`으로 입력받고, `fmt.Printf`로 "10 + 20 = 30" 형태로 출력하라. 0으로 나누는 경우의 에러 처리도 포함하라.

2. **학생 성적표 출력기**: 3명의 학생 이름과 국어, 영어, 수학 점수를 변수에 저장하고, `fmt.Printf`의 너비 지정과 정렬 기능을 활용하여 깔끔한 성적표를 출력하는 프로그램을 작성하라. 각 학생의 합계와 평균도 계산하여 포함하라.

3. **대화형 프로필 생성기**: `fmt.Scanln`과 `bufio.Scanner`를 혼합하여, 사용자에게 이름, 나이, 거주지, 취미, 한 줄 자기소개를 순서대로 입력받고, 입력이 끝나면 보기 좋게 포맷팅된 프로필 카드를 출력하는 프로그램을 작성하라.

4. **영수증 출력 프로그램**: 상품명과 가격 데이터를 변수에 저장하고, `fmt.Printf`를 사용하여 아래와 같은 형태의 영수증을 출력하는 프로그램을 작성하라:
   ```
   ===========================
          OO 마트 영수증
   ===========================
   상품명           단가    수량
   ---------------------------
   사과           1,000원    3
   바나나           500원    5
   우유           2,500원    1
   ---------------------------
   합계:                8,000원
   ===========================
   ```

5. **색상 코드 변환기**: 16진수 RGB 색상 코드(예: 0xFF, 0xAA, 0x33)를 3개의 정수 변수에 저장하고, `fmt.Printf`를 사용하여 10진수, 16진수, 2진수 형태로 각각 출력하는 프로그램을 작성하라. 예를 들어 "R: 255 (0xFF, 0b11111111)" 형태로 출력한다.

## 프로젝트 과제

1. **대화형 설문조사 프로그램**: 사용자에게 5개 이상의 질문(이름, 나이, 직업, 좋아하는 색, 취미 등)을 차례로 물어보고, 모든 입력이 끝나면 `fmt.Sprintf`로 결과 요약 문자열을 만들어 화면에 출력하는 프로그램을 작성하라. 입력 시 `bufio.Scanner`를 사용하여 버퍼 문제를 방지하고, 숫자 입력에는 `strconv` 패키지를 활용하라.

2. **보고서 포맷터**: 여러 줄의 데이터(이름, 숫자, 날짜 등)를 변수에 저장하고, `fmt.Printf`의 다양한 서식 지정자와 너비/정렬 기능을 활용하여 제목, 표, 합계, 날짜 등이 포함된 보고서를 출력하는 프로그램을 작성하라. `fmt.Sprintf`로 각 섹션의 문자열을 먼저 생성한 뒤 최종 출력하는 방식으로 구현하라.
